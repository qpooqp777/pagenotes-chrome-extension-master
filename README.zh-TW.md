# PageNotes

### 在任意網頁上自由新增浮動筆記、標記重點

一款優雅的 Chrome 擴展功能，讓你在任何網頁上自由新增可拖曳的浮動筆記、以 3 種顏色標記重點，並在一處管理所有註釋——**完全儲存在瀏覽器本機**，不上傳至任何伺服器。

[English](README.md) · [简体中文](README.zh-CN.md) · [繁體中文](README.zh-TW.md)

---

## 功能特色

| 功能 | 說明 |
|------|------|
| 📝 浮動筆記 | 在任意網頁上點擊新增可拖曳的優雅筆記卡片 |
| 🎨 3 種顏色標記 | 選取文字 → 選擇黃、綠、粉色標記 |
| 📌 置頂釘選 | 點擊 📌 將筆記釘選在螢幕最上層 |
| 🔄 本機持久儲存 | 所有資料儲存在 IndexedDB，刷新重開依然還原 |
| 📱 Markdown 支援 | 以 Markdown 撰寫筆記，即時預覽格式 |
| 💾 匯出備份 | 隨時將所有資料匯出為 JSON 格式 |
| 🌐 多語言 | 支援繁體中文、簡體中文、English |
| ⌨️ 快捷鍵 | `Ctrl+Shift+N` 快速新增筆記 |
| ☁️ Pro：雲端同步 | 透過 Firebase 跨裝置同步（即將推出）|

---

## 安裝方式

1. 下載或克隆此倉庫
2. 開啟 `chrome://extensions/`
3. 啟用右上角「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇 `pagenotes-chrome-extension` 資料夾

---

## 使用方式

| 操作 | 方法 |
|------|------|
| 新增筆記 | 點擊任意網頁右下角 📝 按鈕 |
| 新增筆記（快捷鍵） | 按 `Ctrl+Shift+N` |
| 從選取範圍新增 | 選取文字 → 工具列點 📝 |
| 標記文字 | 選取文字 → 工具列選擇顏色 |
| 置頂筆記 | 點擊筆記卡片上的 📌 按鈕 |
| 縮小筆記 | 點擊筆記標題列的 — 按鈕 |
| 恢復縮小筆記 | 點 📝 按鈕（顯示紅色數字徽章）→ 選擇要恢復的筆記 |
| 管理所有筆記 | 點擊工具列 📝 圖標 |

---

## 隱私權政策

**你的資料永遠不會離開你的設備。** 所有筆記、標記和設定都儲存在瀏覽器本機的 IndexedDB 中：
- ❌ 無分析工具、無追蹤腳本
- ❌ 無第三方伺服器
- ❌ 無需註冊帳戶

雲端同步（Pro 版）將使用**你自己的 Firebase 帳戶**，只有你能存取資料。
