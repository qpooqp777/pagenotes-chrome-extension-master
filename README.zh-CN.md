# PageNotes

### 在任意网页上自由新增浮动笔记、标记重点

一款优雅的 Chrome 扩展功能，让你在任何网页上自由新增可拖拽的浮动笔记、以 3 种颜色标记重点，并在一处管理所有注释——**完全储存在浏览器本机**，不上传至任何服务器。

[English](README.md) · [繁體中文](README.zh-TW.md) · [简体中文](README.zh-CN.md)

---

## 功能特色

| 功能 | 说明 |
|------|------|
| 📝 浮动笔记 | 在任意网页上点击新增可拖拽的优雅笔记卡片 |
| 🎨 3 种颜色标记 | 选取文字 → 选择黄、绿、粉色标记 |
| 📌 置顶钉选 | 点击 📌 将笔记钉选在屏幕最上层 |
| 🔄 本机持久储存 | 所有资料储存在 IndexedDB，刷新重开依然还原 |
| 📱 Markdown 支持 | 以 Markdown 撰写笔记，即时预览格式 |
| 💾 导出备份 | 随时将所有资料导出为 JSON 格式 |
| 🌐 多语言 | 支持繁体中文、简体中文、English |
| ⌨️ 快捷键 | `Ctrl+Shift+N` 快速新增笔记 |
| ☁️ Pro：云端同步 | 透过 Firebase 跨装置同步（即将推出）|

---

## 安装方式

1. 下载或克隆此仓库
2. 开启 `chrome://extensions/`
3. 启用右上角「开发人员模式」
4. 点击「加载未封装的扩展程序」
5. 选择 `pagenotes-chrome-extension` 文件夹

---

## 使用方式

| 操作 | 方法 |
|------|------|
| 新增笔记 | 点击任意网页右下角 📝 按钮 |
| 新增笔记（快捷键） | 按 `Ctrl+Shift+N` |
| 从选取范围新增 | 选取文字 → 工具列点 📝 |
| 标记文字 | 选取文字 → 工具列选择颜色 |
| 置顶笔记 | 点击笔记卡片上的 📌 按钮 |
| 缩小笔记 | 点击笔记标题列的 — 按钮 |
| 恢复缩小笔记 | 点 📝 按钮（显示红色数字徽章）→ 选择要恢复的笔记 |
| 管理所有笔记 | 点击工具列 📝 图标 |

---

## 隐私权政策

**你的资料永远不会离开你的设备。** 所有笔记、标记和设定都储存在浏览器本机的 IndexedDB 中：
- ❌ 无分析工具、无追踪脚本
- ❌ 无第三方服务器
- ❌ 无需注册账户

云端同步（Pro 版）将使用**你自己的 Firebase 账户**，只有你能存取资料。
