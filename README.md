# PageNotes

### Floating Notes & Highlights for Any Webpage

A beautiful Chrome extension that lets you add floating notes anywhere on any webpage, highlight text in 3 colors, and manage all your annotations in one place — all stored **locally in your browser** with no server.

[繁體中文](README.zh-TW.md) · [简体中文](README.zh-CN.md) · [English](README.md)

---

## Features

| Feature | Description |
|---------|-------------|
| 📝 Floating Notes | Click to add elegant draggable notes on any webpage |
| 🎨 3-Color Highlighting | Select text → choose yellow, green, or pink |
| 📌 Pin & Minimize | Pin notes to stay on top, minimize to tray menu |
| 🔄 Persistent Storage | All data saved in IndexedDB — survives refresh & reload |
| 📱 Markdown Support | Write notes in Markdown with live preview |
| 💾 Export & Import | Export all data as JSON anytime |
| 🌐 Multilingual | Supports Traditional Chinese, Simplified Chinese, English |
| ⌨️ Keyboard Shortcut | `Ctrl+Shift+N` to add a note instantly |
| ☁️ Pro: Cloud Sync | Sync notes across devices via Firebase (coming soon) |

## Installation

1. Download or clone this repository
2. Open `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the `pagenotes-chrome-extension` folder

## How to Use

| Action | How |
|--------|-----|
| Add note | Click the 📝 button on any webpage |
| Add note (shortcut) | Press `Ctrl+Shift+N` |
| Add note from selection | Select text → click 📝 in toolbar |
| Highlight text | Select text → click color in toolbar |
| Pin note | Click the 📌 button on a note card |
| Minimize note | Click — on note card header |
| Restore minimized | Click 📝 FAB (shows badge count) → select note |
| Manage notes | Click the 📝 extension icon |

---

## Privacy

**Your data never leaves your device.** All notes, highlights, and settings are stored locally in your browser using IndexedDB. There are:
- No analytics or tracking scripts
- No third-party servers
- No account required

Cloud sync (Pro) will use **your own Firebase account** — only you have access.
